=======================
一流游戏变速器（正式版）
=======================
可加速或减慢其它程序的运行速度，并且不会改变电脑的时间。

========
使用方法
========
加速：
    Ctrl  + → ：每次加速2^0.001倍
    Shift + → ：每次加速2^0.01倍
    [加速]或→ ：每次加速2^0.1倍
    Alt   + → ：每次加速2^0.5倍

减速：
    Ctrl  + ← ：每次减速2^0.001倍
    Shift + ← ：每次减速2^0.01倍
    [减速]或← ：每次减速2^0.1倍
    Alt   + ← ：每次减速2^0.5倍

恢复正常：
    按[正常]可迅速恢复正常速度；或直接退出程序也可以；

说明：
用鼠标拖动滚动条也可以变速；
→是指右方向键，←是指右方向键；
Ctrl  + →是指按住Ctrl加右方向键；
[加速]或→是指按加速按钮或按右方向键；
2^0.001表示2的0.001次方；


========
注意事项
========
如果你的机器速度比较慢，虽然加速了，但有些游戏的速度反而会变慢；
如果有的游戏不能改变速度，请先变速，再启动游戏试试；
对在服务端有检测的游戏请谨用；
